# Standard Template Library


## what is a Template?
- A blueprint that allows a function or class to work with ANY datatype.

## What are Class Templates?
- Class templates are used for handling data that needs to be remembered (stored) and work for any data type (because of templates).

## What is STL?
- A collection of pre-built containers, algorithms, iterators provided by C++.


## Why do we need STL?
- Funnily enough, even Vectors that we studied earlier is a Class Template. vector is a class that HANDLES dynamic arrays for you. 
- It basically makes programming much much easier and faster by pre-adding a bunch of common data structures(vector, list, stack, queue, map etc..)
- Most STL containers are class templates, so most of them can work with any data type.
- MOST IMPORTANTLY: STL containers are already optimized and are WIDELY used by many devs.
